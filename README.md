# Mein erstes Projekt

Eine einfache Webseite, gebaut mit HTML, CSS und JavaScript.

## Funktionen

- **Responsives Design** – sieht auf allen Geräten gut aus
- **Navigation** – Abschnitte für Start, Über mich und Kontakt
- **Kontaktformular** – mit Eingabevalidierung
- **Smooth Scrolling** – sanftes Scrollen zu den einzelnen Sektionen

## Technologien

- HTML5
- CSS3
- JavaScript (Vanilla)

## Erste Schritte

1. Repository klonen:
   ```bash
   git clone https://github.com/Atlas-KuK/mein-erstes-projekt.git
   ```
2. `index.html` im Browser öffnen.

## Projektstruktur

```
├── index.html   # Hauptseite
├── style.css    # Styling
├── script.js    # Interaktive Funktionen
└── README.md    # Diese Datei
```
