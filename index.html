Einrichtung

1

Google Sheet erstellen

Erstelle ein neues Google Sheet mit folgendem Aufbau:

Datum | Titel | Typ | Pax | Ort | Einkauf-Personal | Koch-Personal | Liefer-Personal | Status | Notizen

Dann: Datei → Im Web veröffentlichen → CSV und die URL unten eintragen.

2

Google Sheet URL eintragen

Die URL bekommst du unter: Datei → Im Web veröffentlichen → CSV-Format auswählen

3

Google Apps Script einrichten

Im Google Sheet: Erweiterungen → Apps Script und den Code aus apps-script/automation.gs einfügen. Dieser:

  • Erstellt automatisch eine PDF mit allen Details
  • Schickt dir eine E-Mail mit der Ordio-Checkliste
  • Aktualisiert das Dashboard
4

Google Form verknüpfen

Erstelle ein Google Form mit denselben Feldern wie das Sheet. Verknüpfe es mit dem Sheet. Dann läuft die Automatisierung bei jedem neuen Auftrag.

!

Ordio – Halbautomatisch

Ordio hat keine öffentliche API. Deshalb bekommst du per E-Mail eine fertige Ordio-Checkliste mit copy-paste-fertigen Texten für:

  • Tagesnotiz (z.B. "Catering 60 Pax – Firma XY")
  • Personalslot Einkauf (Datum, Uhrzeit, Person)
  • Personalslot Kochen
  • Personalslot Lieferung

Das dauert ~2 Minuten in Ordio. Wenn Ordio später eine API freigibt, kann dieser Schritt vollautomatisch werden.
